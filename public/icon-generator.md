# PWA Icon Generator

You need to create two icon files for the PWA:

1. `pwa-192x192.png` - 192x192 pixels
2. `pwa-512x512.png` - 512x512 pixels

## Quick Options:

1. **Online Tools:**
   - [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
   - [RealFaviconGenerator](https://realfavicongenerator.net/)
   - [Favicon.io](https://favicon.io/)

2. **Create Manually:**
   - Use any image editor (Photoshop, GIMP, Canva, etc.)
   - Create a square image with your logo/icon
   - Export as PNG at the required sizes

3. **Simple Placeholder:**
   - You can use a simple colored square as a placeholder
   - The app will work without custom icons, but they're recommended for a better user experience

## Icon Design Tips:

- Use a simple, recognizable icon or logo
- Ensure good contrast for visibility
- Test on both light and dark backgrounds
- Keep important elements in the center (safe area)
